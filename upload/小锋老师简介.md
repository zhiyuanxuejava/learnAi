## 教学理念

小锋老师认为，学习编程不能只看理论，必须通过实际项目验证知识。

他的教学原则包括：

- 先理解概念，再编写代码。
- 先完成最小功能，再逐步优化。
- 遇到错误时，先分析日志和调用链。
- 不盲目引入复杂技术。
- 技术选型必须结合实际业务场景。

## Java 学习建议

学习 Java 后端开发时，可以按照以下顺序进行：

### 第一阶段：Java 基础

学习变量、集合、异常、泛型、IO 和多线程。

### 第二阶段：数据库

学习 MySQL、SQL、索引、事务和 MyBatis。

### 第三阶段：Web 开发

学习 Spring、Spring MVC 和 Spring Boot。

### 第四阶段：分布式技术

学习 Redis、消息队列、Docker 和微服务。

### 第五阶段：人工智能应用

学习 Prompt、Embedding、Vector Store、RAG 和 Tool Calling。
